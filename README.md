-   Easy to use
-   Simple
-   Support
-   MongoDB Support

# Usage

```js
const toanky = require("toanky-invites")
toanky.on(client) // Connects the client to the module (REQUIRED)

client.on("inviteJoin", (user, invite, inviter) => {
// ${user} joined using invite code ${invite.code} from ${inviter.tag}
})
```
